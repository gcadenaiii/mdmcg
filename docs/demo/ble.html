BLE Features Unavailable in Demo

Bluetooth Low Energy (BLE) connectivity requires hardware access and cannot be demonstrated in the online preview mode. This feature is available in the full application.

🔧 BLE Features Include:

  • Device Scanning: Discover nearby BLE sensors
  • Connection Management: Connect and disconnect from devices
  • Real-time Data: Stream live sensor data over Bluetooth
  • Battery Monitoring: Track sensor battery levels
  • Signal Strength: Monitor connection quality

To use BLE features, please download and run the full application on a device with Bluetooth capabilities. The application supports BNO055 sensors and other compatible motion tracking devices.
