Real-Time 3D Visualization

Step Counter & Tracking

Advanced state-machine based step detection with 3D path visualization. Track distance and movement patterns accurately.

Live Data Charts

Real-time graphs for acceleration, orientation, and velocity. Chart.js powered visualizations with smooth updates.

Remote Access

WebSocket streaming allows you to monitor sensors from any device on your network. Perfect for remote projects.

Persistent Calibration

About Sensors Lab

Sensors Lab is a comprehensive Python-based motion tracking and sensor calibration toolkit designed for the BNO055 9-axis IMU sensor running on Raspberry Pi.

The project follows Unix philosophy and separation of concerns, with each tool doing one thing well. From interactive calibration to real-time web visualization, every component is designed to be clear, explicit, and composable.
